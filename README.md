# Electrician Interview Simulator

Practice interview for commercial and residential electricians.

## Features
- Timed sessions with randomized questions.
- Voice-to-text answer input.
- Keyword matching scoring.
- Detailed feedback with NEC references.
- Spaced-repetition study algorithm (coming soon).
- Theory curriculum.

## Database
Contains 500+ interview questions across categories:
- NEC Code
- Electrical Theory
- Practical Knowledge
- Safety
- Troubleshooting
- Management (including Behavioral for Foreman/Superintendent)
- Hands-on Scenarios
